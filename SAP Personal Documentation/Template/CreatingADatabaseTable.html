Creating A Database Table

Add table name to Database table field. Press Create. Database tables created by me will start with either Y or Z depending on the defined namespace.

Add Fields to Database Table

Click the Field tab to begin adding fields. Every SAP database table must contain the field called CLIENT. It is always a key field with the data element MANDT.
To create your own fields, begin by typing the field name column in the next row. SAP database fields differ from other types of databases by abstracting the type of the field by using a data element. Type in the name of the data element to make use of forward navigation.
Fill out the short text field with a description of the data element. Generally, you'll want to create an elementary type with a domain. We'll have to create as domain as well. For consistency, it may be a good idea to name the domain exactly the same as the data element. Before we finish creating the data element, we have to create the domain. Double click the domain name to use forward navigation to create it.
The domain is finally the representation of what the field actually is. Fill out the Short text field to describe what the domain is. Fill out fields Data type, No. characters, Decimal places, and Output length. After filling out the Data type field, a string is populated to the right, which is a signal that the data type is recognized.
While working on the domain, you can specific valid values of the database field using the Value range tab. If a user attempts to enter a value that is not within the specified value range, the user will get an error message. There are three different ways to specify valid values: single values, intervals, and a value table.
Click the save button once you are done. You'll have to activate it to to make use of the domain in your data element. When presented with the activate screen, highlight the domain and press the Continue button as displayed in the screenshot.
You'll need to go back to the data element screen. To do so, press the back button or F3.
On the data element screen, we need to set up the field labels. Go to the Field label tab. Here we can specify what this database field would look like if we wanted to display it on a form. After inserting all of the Field labels, pressing enter in one of the input boxes would automatically populate the length field. Once finished, save and activate the data element.

Set Technical Settings of Database Table

Lastly, we should set up the technical settings of the database table.
Clicking the button on the Data class field opens up another menu. Generally, we want to choose APPL0 (the first option).
